# VIP视频解析服务

这是一个基于Python Flask的VIP视频解析服务，支持多个解析接口，可以解析各大视频网站的VIP视频。

## 功能特点

- 支持多个解析接口自动切换
- 自动检测接口可用性
- 支持移动端访问
- 响应式界面设计
- 支持在线播放

## 安装使用

1. 克隆仓库：
```bash
git clone [你的仓库地址]
cd [仓库名称]
```

2. 安装依赖：
```bash
pip install -r requirements.txt
```

3. 运行服务：
```bash
python web_server.py
```

4. 访问服务：
- 本地访问：http://localhost:5000
- 局域网访问：http://[你的IP]:5000

## 部署说明

本项目支持多种部署方式：
1. 本地运行
2. 云服务器部署
3. PythonAnywhere等平台部署

## 注意事项

- 仅供学习交流使用
- 请勿用于商业用途
- 部分视频可能无法解析，可尝试切换不同接口
